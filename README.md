# Workout Hub

A small, user-focused app to simplify tracking workouts. Easily add the workouts you've completed, view the overall list, and delete ones you no longer need.

## Installation and requirements

- `npm i` to install dependencies
- an .env containing containing a SUPABASE_URL and SUPABASE_PUBLISHABLE_KEY
- a table in the supabase db with the following schema

```sql
 CREATE TABLE IF NOT EXISTS workouts(
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  time_created timestamptz DEFAULT now(),
  exercise text NOT NULL,
  reps smallint NOT NULL,
  workout_id bigint NOT NULL,
  user_id uuid NOT NULL
);

CREATE policy "Enable read access for authenticated users"
ON "public"."workouts"
AS PERMISSIVE
FOR SELECT
TO authenticated
USING (
  true
);

CREATE policy "Enable insert for authenticated users only"
ON "public"."workouts"
AS PERMISSIVE
FOR INSERT
TO authenticated
WITH CHECK (
  true
);

CREATE policy "Enable delete for authenticated users only"
ON "public"."workouts"
AS PERMISSIVE
FOR DELETE
TO authenticated
USING (
  true
);
```

- `npm run dev` to begin the site after prerequisites

## Usage

Either browse through the publicly available exercises, or login to begin recording your workouts.

After logging in, the dashboard offers access to adding workouts, viewing previous workouts, and removing unneeded ones.
